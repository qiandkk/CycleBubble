# CycleBubble

一款面向女性的情绪理解工具，围绕月经周期做情绪可视化，帮助用户理解自己的情绪变化。

## 设计理念

- **不做医学诊断**，只做趋势提示
- **减少标签化**，不把情绪判断成对错
- **不做好友/私信**，只做匿名内容回应
- **优先隐私安全**，最小化数据采集

## 三个核心功能

### 1. 周期泡泡
用可视化泡泡表达周期阶段和情绪倾向。泡泡的颜色、大小和概率反映当前可能处于的阶段。

### 2. 情绪日记 + 温和反馈
便签式轻量记录事件和感受，给出非评判式的温和反馈。

### 3. 匿名共鸣卡片
基于语义相似度，把经历相似的匿名表达做成温和回应卡片，不暴露身份。
