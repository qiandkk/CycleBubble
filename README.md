# CycleBubble

CycleBubble 是一款帮助女性理解周期与情绪关系的产品。它不是经期记录工具、情绪治疗工具或冥想产品，而是通过周期状态、情绪记录和匿名共鸣，帮助用户理解“最近的情绪可能从哪里来”。

## 项目定位

CycleBubble 的核心价值：

- 帮助用户理解周期与情绪之间的关系
- 用温和、克制的方式呈现当前周期状态
- 通过个人记录线索解释情绪波动来源
- 通过匿名共鸣提供被理解感

## 设计方向

当前首页设计方向已经从概念展示页调整为真实产品首页：

- 用户第一眼看到自己的周期数据
- 首页弱化品牌宣传和理念展示
- 优先呈现当前周期状态、情绪倾向解释、今日关注点和个人记录线索
- 避免科技感、赛博朋克、AI 概念图和冥想 App 化表达
